Fund Address with Faucet

The Faucet allows you to fund test addresses. We offer two different options for testing your blockchain application: Bitcoin Testnet3, and BlockCypher’s Test Chain. We offer automated faucets for both Testnet3 and BlockCypher’s Test Chain. Read more info about testing in our docs.

API called: '/v1/btc/test3' or '/v1/bcy/test'
