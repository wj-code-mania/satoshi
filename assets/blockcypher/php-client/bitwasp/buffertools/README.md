# Buffertools
=============
This library provides a `Buffer` and `Parser` class to make dealing with binary data in PHP easier.

## Requirements:
 * PHP 5.4+
 * Composer
 * ext-gmp

##Installation
You can install this library via Composer: `composer require bitwasp/buffertools` or 
```
{
    "require": {
        "bitwasp/buffertools": "~0.1"
    }
}
```
